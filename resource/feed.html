我拥有大概10MH/s的算力，曾经也想过自建矿池，也用过哈希宝这类小白产品，最终选择了c3pool。我将在下面详细说明为何做这个选择。

接下来，我会介绍如何创建钱包，选择矿池，挖矿到交易所交易的整个流程。

挖矿流程

假设你已有设备，那么一般的挖矿流程是 创建钱包 → 选择矿池 → 挖矿 → 提现到钱包 → 交易所交易 ，经过这一套流程，就能将加密货币兑换为法币（法定货币，即人民币）。

这里还有另一种选择，就是使用哈希宝的这类小白（傻瓜式）产品，把所有流程精简为 注册 → 挖矿 → 提现 。

假如你的算力非常非常的低，或者怕麻烦，那么哈希宝可能适合你。交易所都要求最低提现20个USDT（约等于130元），各种买家也鱼龙混杂，对于小额多笔交易，冻结银行卡的风险会更大。

了解小白产品
哈希宝（类似产品还有“一路赚钱”，更黑）
优点：

  1. 傻瓜式挖矿软件，操作简单。提现方便快捷。
  2. 至于有些人说的哈希宝偷算力，我用CPU来挖倒是没遇到。实测收益比全球第一大矿池 minexmr.com 还要高少许。
  3. 公司为成都区块通科技有限公司，注册资本500万。跑路风险较低。
  4. 平台被攻击会很快恢复，毕竟这么大个公司摆在这。

缺点：

  1. 提现手续费巨高。假设平台币“哈希”为1500元，提现15-30元需要手续费20%，三千到一万五需要手续费2%，七万五以上1%。真能挖到七万五，去哪家交易所不好？
  2. 客服态度比较差，不回消息是常态。这点有人也在知乎提到。

了解这类产品了吗？接下来看看一般的挖矿流程。

创建钱包

我选择门罗币的钱包。有两种最常用的方式，一种使用官方钱包，需要同步高达100G的区块。另一种是使用MyMonero钱包，由MyMonero服务器同步区块。安全性肯定不如官方钱包了，但是门罗币比较少的话还是可以用的。几百个门罗币还是用官方钱包吧。MyMonero的转账手续费折合人民币0.1元/次，它帮你同步了100G的区块，这点手续费可以接受。

为了尽量提高安全性，我的方法是创建一个虚拟机，里面安装Windows 10 LTSC，用命令行连接KMS激活，以后就用这个虚拟机专门做交易。

前往MyMonero官网，下载Windows版本。创建钱包后把助记词手写到一张纸，千万别直接复制保存在任何一个地方。若想用手机来查看钱包，比较麻烦，它没有扫描二维码连接钱包的功能。我用的是安卓版本，助记词不能输入25位的中文，只能复制钱包地址，然后分别手打长达64位的查看密钥、支付密钥，打完后连不上服务器... CAO，我在电脑上直连是可以的，早知道我先开个VPN。心态崩了，放弃！

（为什么不能复制？因为联网后就不安全了。即使是不联网状态下复制，完成后删除手机剪贴板，但你不知道是XX软件的某个员工还是黑客已经记录下来，联网时向服务器发送。）

选择矿池

在这个网站可以查看热门的XMR矿池列表。我最开始使用了minexmr.com，抽水1%。因为我的算力比较高，后来我想自建矿池。去Github搜了几个矿池项目，也许是我技术太差，没有一个能搭建起来的。这时看到了c3pool，说是0抽水，我就随便放了个机器在里面试试。这一试不要紧，我发现官网上显示的算力比 minexmr.com 以及哈希宝（用的是f2pool）高了30%-100%！这不会是假的吧。我就测试了几天，把币提到钱包，最后得出结论：c3pool牛逼！

根据c3pool的介绍，这个矿池会在多种算法的多个币中，选择利润最高的币去挖，因此收益高于只挖一个币的传统矿池。这，我100%选择c3pool啊。

c3pool（猫池）（注：本文写于猫池改版前）
优点：

  1. 同样的机器，收益会高出30%-100%。
  2. 客服态度也太好了吧。前面提到的哈希宝客服态度有多差，猫池的客服态度就有多好，真是鲜明的对比啊！发消息，问问题基本都是秒回。猫池老板说整个矿池就他一个技术兼客服。他仿佛24小时坐在电脑前一样（当然，仅限他不忙的时候）。

  1. 项目为个人兴趣使然。能不能坚持下去，完全靠猫总（猫池老板）个人。他说公司的业务不涉及矿池，这个矿池完全是自己的兴趣。
  2. 经常被攻击。3月初的攻击导致每个矿池节点都出问题，每天断几个小时。对普通人来说就是几个小时的电费损失，对矿老板来说，要是没有及时切换矿池的话损失就大了。猫总说只要有收益被影响都会给补偿。不过也不是每次都有，也许因为这不是猫池导致的问题。3月20几号也有一次攻击，但这一次猫总补偿了很多。最近矿池节点全部掉线的情况没有了，并且世界每个区域都有几个矿池服务器。注意：假如矿池被攻击了，马上换这个同款矿池，因为补偿不是每次都有。有1MH以上的算力也可以问猫总要专线，不怕被攻击。
  3. 可能是被攻击导致稳定性不太好。我以前用AWS来挖矿，在后台看到这样的统计：

挖矿

c3pool官网有一键挖矿脚本，直接用就好了。有什么问题问客服。

说说我遇到的问题：使用官网提供的一键脚本，挖矿时CPU占用率只有50%。我通过修改配置，让CPU占用100%，但是收益却减半了。这种情况多发生在英特尔的处理器上。因为处理器的全部核心已经用完，剩下的50%是超线程技术。让CPU同时处理两倍的工作量，效率就下降了。

注意：计算收益时，不要看挖矿软件显示的算力，不同币种不同算法下的算力没有任何可比性。算力以矿池网站上显示的为准，并且网站也有预估日收入功能（实际收入没有那么多）。

提现到钱包

在c3pool可以设置自动提现金额，最小0.003门罗币。有时候达到了限额也没有支付，因为支付队列太多人了。不用担心，迟早会付的。

交易所交易

这里我选择的是币安。因为我看了各种网站，包括知乎、贴吧，大家都说火币冻卡风险大。也有相反的观点，说每个交易所都有冻卡风险，火币冻卡风险大是因为它是最大的交易所。既然都有冻卡风险，那就选择交易手续费最低的币安吧。币币交易只需手续费0.1% 。

交易所交易又分为几个流程：充值门罗币到现货账户 → 将XMR换成稳定币USDT（币币交易） → 到自选区选择合适买家交易（法币交易） 。

注册币安和充值XMR

首先注册币安，通过这个推荐链接注册可以在每笔币币交易时获得20%的返现。据说还有40%返现的注册链接，但对邀请人的要求比较高。我到处找没找到。Google Play商店很多人在币安APP下评论40%返现的链接，没一个是真的。

注册后完成实名验证，身份证注意打上“仅限币安实名认证”的水印，然后再完成人脸识别。

然后点击右上角的钱包，选择现货账户，搜索XMR，选择充值。

右边就会出现一个供你充值的钱包地址。从你的钱包转账到这个地址，5分钟左右到账。

既然要将门罗币充值到币安，那为什么不直接在矿池填写币安的地址呢？因为那个地址并非一定不变，交易所也可能会暂停XMR的充币和提币业务。

我需要交易20个XMR。我比较谨慎，先转0.0001试试，4分钟后到账。然后想把剩下的19.9999也转过来，但钱包显示1.5999被锁定。我发现锁定的金额正好是一次收款的金额。这...支付宝转账一块钱，你不能锁我一万块啊... 这可能是从那个区块中拿出0.0001，需要等待区块同步。但是我明明有更小的区块。我就先把18.4转过去了。成功后发现剩下的1.5999解锁了，那就转过去吧。

XMR兑换USDT（币币交易）

收到转账后，回到现货账户，点击交易，选择XMR/USDT

进入到交易页面

参考当前的市价和价格走势，把价格稍微标高一点。比如市价221.22，可以标222.00。不急卖的话可以再高点。不要用市价交易，比较亏。

我在晚上标了个223.00，结果到第二天都没人交易。改为222.22后几分钟就卖出了

法币交易

从右上角的钱包菜单中，找到C2C账户，在USDT栏目选择“划转”，从现货账户到C2C账户。

从右上角的个人菜单中，找到账户安全，点击左边的收款方式，添加你不常用的收款方式。千万别用常用支付宝、微信、银行卡来收款。万一被冻结了就麻烦了。使用一个不常用的银行卡，小号支付宝/微信。冻卡是小概率事件，就算真的冻卡了，一般也不会冻结本人名下的所有支付方式，所以这样做是有意义的，不至于导致你原有的钱也用不了。

完成后点击“卖”，就可以选择买家进行交易了。这一步一定要在手机上进行，因为只有币安APP可以查看买家的注册时间、首次交易时间。绝大多数账户都是最近3个月才注册的新号，有一定安全风险。尽量选择交易时间久的账户，因为假如有卖家收到黑钱被冻卡，买家的号应该早被封了。当时最高价6.65，我看那个买家是才注册不到半个月的号，有可能就是奔着洗钱来的。接下来标6.64的账号都比较新，我就选择了6.63，离首次交易时间有半年以上的一个买家。我认为为了安全牺牲一些钱是值得的。

买家不到3分钟就把钱转给我了，这里务必核对买家名字和转账过来的人的名字是否相同。核对名字和金额，确认无误后点击“确认收款”。

买家还发了一条短信提醒我

整个交易流程非常迅速，丝毫没有拖泥带水。收入29436元。

XMR当时价格是1437元，假如直接换成人民币的话是28740元。但是经过交易所换成USDT再换成人民币，最后收入29436元，多了696元。

至此，我的第一次交易就完成了。
有人说收款后马上把钱领出来，然后再存入另一张银行卡，可以切断资金的联系。但也有人说这样会加大洗钱的嫌疑。我打算直接拿来花，不打算领出来。

为什么不选择私下交易

不经过平台，直接在挖矿群中喊一声“有谁收门罗币”。确实可行，但是风险不见得会低。

  1. 你怎么知道买家的钱是干净的？问他要收支记录吗？你一样可以在交易所发卖币的广告，同样可以要求买家给收支记录。
  2. 你先转币还是对方先转钱？这种交易只有一方有保障。交易所相当于一个担保人的身份，确保你不会被骗。有人在闲鱼卖币，币转过去了，但是直接被拉黑。又或者钱过去了，币却没到。
  3. 你按照什么样的价格卖给买家？直接把XMR换算成人民币的金额与XMR换USDT再换人民币相比会有波动，往往是换USDT再换人民币价格要高。

当然，假如能线下当面交易的话可能会更好，双方当面沟通。

写在最后

选择一个合适的矿池，一个合适的交易所，一个合适的买家很重要。会不会有人说，最重要的可能是有10MH的算力。嗯，这个确实重要。能获得这个算力纯属巧合了，掌管着一些小型超算。

后记

我3月底卖出时是1437元。现在是4月底，短短一个月，门罗币已经涨到了2309元。别卖了，囤起来吧！

如何使用 AWS Organizations

登录账户后，在右上角点击你的AWS名称，选择“My Organization”。

官方文档

以下所有的“服务抵扣金额”、“积分”、“优惠码”，均指“credits”。
在AWS文档的 AWS 账单和成本管理 部分针对AWS Organizations 的整合账单“跨单个和多个账户应用 AWS 服务抵扣金额”的规定，给出了这样的描述：

  • 账单周期从每月的第一天开始。

  • 如果账户在某月的第一天属于某个不属于某个组织的个人，但在当月晚些时候加入组织，则 AWS 会将该个人拥有的积分应用到该个人针对该月的使用量的账单。下个月，AWS 将服务抵扣金额应用到加入组织。

  • 如果账户在月份的第一天属于某个组织，则 AWS 会将付款人账户或任何关联账户兑换的服务抵扣金额应用于组织的账单，而不论账户所有者在当月晚些时候是否离开组织。

  • 如果个人在某个月内离开组织，AWS 将从下个月的第一天开始针对个人账户应用服务抵扣金额。

  • 服务抵扣金额在当月的任何时刻与加入组织的所有账户共享。但是，组织的共享积分池仅包含自该月第一天以来已成为组织的一部分的账户的积分。

“如果账户在某月的第一天属于某个不属于某个组织的个人”，AWS酱的机器翻译真是神奇，原句是“If an account is owned on the first day of the month by an individual who is not part of an organization”，翻译过来是“如果一个账户由当月第一天不属于某个组织的个人拥有”，或者“如果一个账户当月第一天属于个人，不属于组织”。

查阅了各种文档，并且根据个人的使用经验总结 AWS Organizations 的部分规定和限制如下：

  • 独立账户在当月加入组织，优惠码不会共享进组织的积分池，次月1号起优惠码才会与组织共享。
  • 组织中的账户若离开组织，优惠码在当月仍与组织共享，次月1号才取消共享。
  • 组织内的成员数量默认最多为4个。（我的组织已经添加了6个账户还能继续添加，看来AWS文档中写的默认配额是错误的）

  • 组织管理员创建无需信用卡的新账户时，填写的邮箱会收到可用免费套餐的邮件，实际上不适用免费套餐。
  • 若有多个适用免费套餐的账户加入组织，免费套餐会共享但不会叠加。
  • 无需信用卡的新账户每个区默认配额为8或32。（默认显示为5，代表该区域未解锁。开一台机器，配额刷新后为8或32）

在组织的积分池中，使用积分的优先级如下：

  1. 即将过期

  2. 最小数量的适用产品

  3. 最旧的服务抵扣金额

  2. 具有最高支出的账户

  4. 在该服务内具有最高支出的产品
